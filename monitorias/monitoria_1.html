1 Monitoria 1

Nessa primeira monitoria, vamos revisar tudo o que vimos até aqui: os principais tipos de funções de uma variável real — suas propriedades, domínios, imagens e gráficos — e as regras de derivação, com ênfase nas aplicações econômicas. O objetivo é consolidar os conceitos antes de avançarmos para o cálculo integral e as funções de várias variáveis.

1.1 Revisão de Funções

1.1.1 Tipos de Funções

Apresentamos abaixo os principais tipos de funções que aparecem ao longo do curso, com suas características essenciais.

Função Linear: \(f(x) = ax + b\)

  • Gráfico: reta com inclinação \(a\) e intercepto \(b\).
  • Domínio: \(\mathbb{R}\). Imagem: \(\mathbb{R}\) (se \(a \neq 0\)) ou \(\{b\}\) (se \(a = 0\)).
  • Em Economia: funções de custo, demanda e oferta lineares.

Função Quadrática: \(f(x) = ax^2 + bx + c\), \(a \neq 0\)

  • Gráfico: parábola côncava para cima (\(a>0\)) ou para baixo (\(a<0\)).
  • Vértice em \(x_v = -b/(2a)\); raízes via fórmula de Bhaskara.
  • Em Economia: funções de lucro, receita total.

Função Polinomial: \(f(x) = a_n x^n + \cdots + a_1 x + a_0\)

  • Generalização das anteriores para grau \(n\) arbitrário.
  • Domínio: \(\mathbb{R}\). O comportamento nos extremos depende do grau e do coeficiente líder \(a_n\).
  • Em Economia: funções de custo total com múltiplas inflexões.

Função Exponencial: \(f(x) = a^x\), \(a > 0\), \(a \neq 1\)

  • Domínio: \(\mathbb{R}\). Imagem: \((0, +\infty)\).
  • \(a > 1\): crescente; \(0 < a < 1\): decrescente. Sempre positiva.
  • Em Economia: crescimento composto, descontos, modelos de juros.

Função Logarítmica: \(f(x) = \log_a x\), \(a > 0\), \(a \neq 1\)

  • Domínio: \((0, +\infty)\). Imagem: \(\mathbb{R}\).
  • Inversa da exponencial: \(\log_a(a^x) = x\) e \(a^{\log_a x} = x\).
  • Em Economia: funções de utilidade (log-utilidade), elasticidades.

1.2 Revisão de Derivadas

1.2.1 Conceito

A derivada de \(f\) em \(x_0\) é a taxa de variação instantânea — a inclinação da reta tangente ao gráfico naquele ponto:

\[f'(x_0) = \lim_{h \to 0} \frac{f(x_0 + h) - f(x_0)}{h}.\]

1.2.2 Regras de Derivação

Regra da Potência: \(\dfrac{d}{dx} x^n = nx^{n-1}\)

Soma e Diferença: \([f(x) \pm g(x)]' = f'(x) \pm g'(x)\)

Produto: \([f(x) \cdot g(x)]' = f'(x)\,g(x) + f(x)\,g'(x)\)

Quociente: \(\left[\dfrac{f(x)}{g(x)}\right]' = \dfrac{f'(x)\,g(x) - f(x)\,g'(x)}{[g(x)]^2}\)

Cadeia: \([f(g(x))]' = f'(g(x)) \cdot g'(x)\)

1.3 Exercícios Resolvidos

1.3.1 Derivadas de Funções Polinomiais

Exemplo 1. Calcule \(g'(x)\) para \(g(x) = 3x^4 - 2x^3 + 5x^2 - 7x + 1\).

Aplicando a regra da potência e da soma termo a termo:

\[g'(x) = 12x^3 - 6x^2 + 10x - 7.\]

Exemplo 2. Seja \(g(x) = \dfrac{x^3 + 2x}{x}\) para \(x \neq 0\). Simplifique e derive.

\[g(x) = x^2 + 2 \Rightarrow g'(x) = 2x.\]

1.3.2 Regra do Produto

Exemplo 3. \(f(x) = (x^2 + 1)(3x - 5)\).

\[f'(x) = 2x(3x-5) + (x^2+1)\cdot 3 = 6x^2 - 10x + 3x^2 + 3 = 9x^2 - 10x + 3.\]

Exemplo 4. \(f(x) = x^3 e^x\).

\[f'(x) = 3x^2 e^x + x^3 e^x = x^2 e^x(3 + x).\]

1.3.3 Regra do Quociente

Exemplo 5. \(f(x) = \dfrac{x^2 + 3x}{2x - 1}\).

\[f'(x) = \frac{(2x+3)(2x-1) - (x^2+3x)\cdot 2}{(2x-1)^2} = \frac{4x^2+4x-3-2x^2-6x}{(2x-1)^2} = \frac{2x^2-2x-3}{(2x-1)^2}.\]

Exemplo 6. \(f(x) = \dfrac{\ln x}{x}\).

\[f'(x) = \frac{\frac{1}{x} \cdot x - \ln x \cdot 1}{x^2} = \frac{1 - \ln x}{x^2}.\]

1.3.4 Regra da Cadeia

Exemplo 7. \(f(x) = (3x^2 + 5)^4\).

Com \(u = 3x^2+5\):

\[f'(x) = 4(3x^2+5)^3 \cdot 6x = 24x(3x^2+5)^3.\]

Exemplo 8. \(f(x) = \sqrt{x^2 + 1} = (x^2+1)^{1/2}\).

\[f'(x) = \frac{1}{2}(x^2+1)^{-1/2} \cdot 2x = \frac{x}{\sqrt{x^2+1}}.\]

Exemplo 9. \(f(x) = \ln\!\left(\dfrac{\sqrt{2x+1}}{3x-2}\right)\).

Usando propriedades do logaritmo:

\[f(x) = \tfrac{1}{2}\ln(2x+1) - \ln(3x-2).\]

\[f'(x) = \frac{1}{2} \cdot \frac{2}{2x+1} - \frac{3}{3x-2} = \frac{1}{2x+1} - \frac{3}{3x-2}.\]

Exemplo 10. \(f(x) = e^{\sqrt{x^2 - 5x}}\).

Com \(u = \sqrt{x^2-5x} = (x^2-5x)^{1/2}\):

\[f'(x) = e^{\sqrt{x^2-5x}} \cdot \frac{2x-5}{2\sqrt{x^2-5x}}.\]

1.3.5 Aplicações: Estudo de Funções

A derivada permite analisar o comportamento de \(f\) de forma sistemática:

  • Crescimento/Decrescimento: \(f'(x) > 0 \Rightarrow f\) crescente; \(f'(x) < 0 \Rightarrow f\) decrescente.
  • Pontos críticos: \(f'(x) = 0\) ou \(f'(x)\) não existe — candidatos a máximos e mínimos.
  • Concavidade: \(f''(x) > 0 \Rightarrow\) convexa; \(f''(x) < 0 \Rightarrow\) côncava.
  • Ponto de inflexão: \(f''(x) = 0\) com mudança de sinal.

Exemplo 11. Analise \(f(x) = x^3 - 6x^2 + 9x + 1\).

\[f'(x) = 3x^2 - 12x + 9 = 3(x-1)(x-3).\]

Pontos críticos: \(x = 1\) (máximo local) e \(x = 3\) (mínimo local).

\[f''(x) = 6x - 12 \Rightarrow f''(1) = -6 < 0 \ (\text{máximo}),\quad f''(3) = 6 > 0 \ (\text{mínimo}).\]

Ponto de inflexão: \(f''(x) = 0 \Rightarrow x = 2\), com \(f(2) = 3\).

library(ggplot2)
library(patchwork)

x <- seq(-0.5, 4.5, length.out = 400)
base <- theme_minimal(base_size = 12)

df <- data.frame(
  x   = x,
  f   = x^3 - 6*x^2 + 9*x + 1,
  fp  = 3*x^2 - 12*x + 9,
  fpp = 6*x - 12
)

pts <- data.frame(
  x     = c(1, 2, 3),
  y     = c(5, 3, 1),
  label = c("Máx. local\n(1, 5)", "Infl.\n(2, 3)", "Mín. local\n(3, 1)"),
  tipo  = c("Máximo", "Inflexão", "Mínimo")
)

p1 <- ggplot(df, aes(x, f)) +
  geom_line(color="#440154", linewidth=1.1) +
  geom_hline(yintercept=0, color="gray70") +
  geom_point(data=pts, aes(x,y,color=tipo), size=3) +
  geom_text(data=pts, aes(x,y,label=label,color=tipo),
            vjust=-0.6, size=3, fontface="bold") +
  scale_color_manual(values=c("Máximo"="#FDE725","Inflexão"="#31688e","Mínimo"="#35b779")) +
  labs(title=expression(f(x)==x^3-6*x^2+9*x+1), x="x", y="f(x)", color=NULL) +
  theme_minimal(base_size=12) + theme(legend.position="bottom")

p2 <- ggplot(df, aes(x, fp)) +
  geom_line(color="#31688e", linewidth=1) +
  geom_hline(yintercept=0, linetype="dashed", color="gray50") +
  geom_vline(xintercept=c(1,3), linetype="dashed", color="gray50") +
  labs(title=expression(f*minute(x)==3*x^2-12*x+9), x="x", y="f'(x)") + base

p1 + p2

1.3.6 Aplicação Econômica: Otimização de Lucro

Exemplo 12. Uma firma tem receita total \(R(q) = 50q - q^2\) e custo total \(C(q) = q^2 + 10q + 5\). Encontre a quantidade que maximiza o lucro.

O lucro é:

\[\pi(q) = R(q) - C(q) = 50q - q^2 - q^2 - 10q - 5 = -2q^2 + 40q - 5.\]

Condição de primeira ordem:

\[\pi'(q) = -4q + 40 = 0 \Rightarrow q^* = 10.\]

Condição de segunda ordem: \(\pi''(q) = -4 < 0\) — confirma máximo.

\[\pi(10) = -200 + 400 - 5 = 195.\]

library(ggplot2)

q <- seq(0, 22, length.out = 300)
df <- data.frame(q=q, lucro=-2*q^2+40*q-5)

ggplot(df, aes(q, lucro)) +
  geom_line(color="#440154", linewidth=1.1) +
  geom_vline(xintercept=10, linetype="dashed", color="#31688e") +
  geom_point(aes(x=10, y=195), color="#FDE725", size=4) +
  annotate("text", x=11.5, y=185, label="q* = 10\nπ = 195",
           color="#31688e", size=3.8) +
  geom_hline(yintercept=0, color="gray70") +
  labs(x="q (quantidade)", y=expression(pi(q)),
       title="Função Lucro") +
  theme_minimal(base_size=13)
