## Tópicos Cobertos

- Definição de funções de várias variáveis reais
- Função de produção Cobb-Douglas: retornos de escala e capital fixo
- Curvas de nível: definição, isoquantas e exemplos geométricos
- Exemplos: $f(x,y) = x^2 + y^2$, $f(x,y) = x^2 - y^2$, $f(x,y) = \frac{1}{x - y^2}$, $f(x,y) = \frac{1}{x^2 + y^2}$
