1 Limites e Continuidade

1.1 Motivação Econômica

O conceito de limite é a fundação matemática de toda análise marginal em Economia. Intuitivamente, o custo marginal é o que acontece com o custo total quando produzimos uma unidade infinitamente pequena a mais — ou seja, é o limite da variação média quando o incremento de quantidade tende a zero:

\[CMg(q) = \lim_{\Delta q \to 0} \frac{C(q + \Delta q) - C(q)}{\Delta q} = C'(q).\]

Outros exemplos onde limites aparecem de forma central:

  • Equilíbrio de mercado como limite de ajustamento. Em modelos de ajustamento de preços tipo Walras, o preço converge para o equilíbrio \(p^*\) à medida que o tempo tende ao infinito: \(\lim_{t \to \infty} p(t) = p^*\). A existência desse limite é condição de estabilidade do equilíbrio.

  • Continuidade como pressuposto comportamental. Funções de demanda e utilidade são geralmente supostas contínuas: pequenas variações de preço ou renda não geram saltos na demanda. A descontinuidade teria implicações fortes (por exemplo, mercados que “somem” com variações de preço infinitesimais).

  • Assíntotas e mercados monopolistas. A receita marginal de um monopólio com demanda \(P(q)\) pode tender a \(-\infty\) quando \(q\) cresce sem limite — indicando que aumentar indefinidamente a oferta destrói a receita.

1.2 Limite Lateral

Dado um ponto \(b \in \mathbb{R}\), podemos nos aproximar dele pela direita (\(x \to b^+\)) ou pela esquerda (\(x \to b^-\)). Esses dois limites podem diferir.

Limite pela direita: \(\displaystyle\lim_{x \to b^+} f(x) = L\) — valores de \(x\) maiores que \(b\), se aproximando de \(b\).

Limite pela esquerda: \(\displaystyle\lim_{x \to b^-} f(x) = M\) — valores de \(x\) menores que \(b\), se aproximando de \(b\).

Se \(L \neq M\), os limites laterais existem mas são distintos, e o limite bilateral \(\lim_{x \to b} f(x)\) não existe.

Exemplo. Considere \(f(x) = \begin{cases} 3 + x & \text{se } x \leq 1 \\ 3 - x & \text{se } x > 1 \end{cases}\).

\[\lim_{x \to 1^-} f(x) = 4, \qquad \lim_{x \to 1^+} f(x) = 2.\]

Como \(4 \neq 2\), o limite em \(x = 1\) não existe, embora \(f(1) = 4\).

1.3 Definição de Limite

O limite bilateral existe se e somente se os dois limites laterais existem e coincidem:

\[\exists\, \lim_{x \to b} f(x) \iff \lim_{x \to b^+} f(x) = \lim_{x \to b^-} f(x).\]

Nesse caso, o valor comum é o limite: \(\lim_{x \to b} f(x) = L\).

Observação. O limite descreve o comportamento de \(f\) nas vizinhanças de \(b\), independentemente do valor de \(f(b)\) — ou mesmo de \(f(b)\) existir.

library(ggplot2)
library(patchwork)

base <- theme_minimal(base_size = 11)

# Exemplo 1: limites laterais distintos
x1a <- seq(-1, 1,  length.out = 200)
x1b <- seq(1,  3,  length.out = 200)
df1 <- rbind(
  data.frame(x = x1a, y = 3 + x1a, ramo = "x ≤ 1"),
  data.frame(x = x1b, y = 3 - x1b, ramo = "x > 1")
)
pts1 <- data.frame(
  x     = c(1, 1),
  y     = c(4, 2),
  tipo  = c("fechado", "aberto")
)

p1 <- ggplot(df1, aes(x, y)) +
  geom_line(aes(color = ramo), linewidth = 1) +
  geom_point(data = pts1[1,], aes(x,y), color = "#440154", size = 3) +
  geom_point(data = pts1[2,], aes(x,y), color = "#440154", size = 3,
             shape = 21, fill = "white", stroke = 1.5) +
  geom_hline(yintercept = 0, color = "gray80") +
  geom_vline(xintercept = 0, color = "gray80") +
  scale_color_manual(values = c("#440154","#35b779")) +
  labs(title = "Limites laterais distintos", x = "x", y = "f(x)", color = NULL) +
  base + theme(legend.position = "bottom")

# Exemplo 2: limite bilateral existente mas f(2) ≠ L
x2a <- seq(-1, 3, length.out = 400)
y2a <- 3 + x2a
pts2 <- data.frame(x = c(2, 2), y = c(5, 2), tipo = c("aberto","fechado"))

p2 <- ggplot(data.frame(x = x2a, y = y2a), aes(x, y)) +
  geom_line(color = "#31688e", linewidth = 1) +
  geom_point(data = pts2[1,], aes(x,y), color = "#31688e", size = 3,
             shape = 21, fill = "white", stroke = 1.5) +
  geom_point(data = pts2[2,], aes(x,y), color = "#FDE725", size = 3) +
  geom_hline(yintercept = 0, color = "gray80") +
  geom_vline(xintercept = 0, color = "gray80") +
  annotate("text", x = 2.4, y = 5, label = "lim = 5", color = "#31688e", size = 3.5) +
  annotate("text", x = 2.4, y = 2, label = "f(2) = 2", color = "#FDE725", size = 3.5) +
  labs(title = "Limite existe, mas lim ≠ f(a)", x = "x", y = "f(x)") +
  base

p1 + p2

1.4 Limites Infinitos e no Infinito

1.4.1 Limite Infinito

Dizemos que \(\lim_{x \to b} f(x) = +\infty\) quando \(f(x)\) cresce sem limite à medida que \(x \to b\). Nesse caso, a reta \(x = b\) é uma assíntota vertical do gráfico de \(f\).

Exemplo. Para \(f(x) = \dfrac{1}{x}\), temos \(\mathbb{D} = \mathbb{R}^*\) e:

\[\lim_{x \to 0^+} f(x) = +\infty, \qquad \lim_{x \to 0^-} f(x) = -\infty.\]

A reta \(x = 0\) é assíntota vertical; como os limites laterais divergem, o limite em \(x = 0\) não existe.

1.4.2 Limite no Infinito

Quando \(x\) cresce (ou decresce) sem limite, \(f(x)\) pode convergir para um valor \(L\). Nesse caso, \(y = L\) é uma assíntota horizontal.

Para \(f(x) = \dfrac{1}{x}\):

\[\lim_{x \to +\infty} f(x) = 0^+, \qquad \lim_{x \to -\infty} f(x) = 0^-.\]

A reta \(y = 0\) é assíntota horizontal.

Exemplo com deslocamento. Para \(f(x) = \dfrac{1}{x-2} + 2\):

  • Assíntota vertical em \(x = 2\) (denominador nulo).
  • Assíntota horizontal em \(y = 2\) (limite no infinito).
library(ggplot2)
library(patchwork)

base <- theme_minimal(base_size = 11)

# f(x) = 1/x
x_neg <- seq(-4, -0.05, length.out = 300)
x_pos <- seq(0.05, 4,   length.out = 300)
df_hip <- rbind(
  data.frame(x = x_neg, y = 1/x_neg),
  data.frame(x = x_pos, y = 1/x_pos)
)

p1 <- ggplot(df_hip, aes(x, y)) +
  geom_line(color = "#440154", linewidth = 1) +
  geom_vline(xintercept = 0, linetype = "dashed", color = "tomato") +
  geom_hline(yintercept = 0, linetype = "dashed", color = "steelblue") +
  coord_cartesian(ylim = c(-6, 6)) +
  annotate("text", x = 0.5, y = 5.2, label = "A.V.: x = 0", color = "tomato",   size = 3.2) +
  annotate("text", x = 3.0, y = 0.7, label = "A.H.: y = 0", color = "steelblue", size = 3.2) +
  labs(title = "f(x) = 1/x", x = "x", y = "f(x)") + base

# f(x) = 1/(x-2) + 2
xn2 <- seq(-2, 1.85, length.out = 300)
xp2 <- seq(2.15, 6,  length.out = 300)
df2 <- rbind(
  data.frame(x = xn2, y = 1/(xn2 - 2) + 2),
  data.frame(x = xp2, y = 1/(xp2 - 2) + 2)
)

p2 <- ggplot(df2, aes(x, y)) +
  geom_line(color = "#35b779", linewidth = 1) +
  geom_vline(xintercept = 2, linetype = "dashed", color = "tomato") +
  geom_hline(yintercept = 2, linetype = "dashed", color = "steelblue") +
  coord_cartesian(ylim = c(-3, 7)) +
  annotate("text", x = 2.7, y = 6.2, label = "A.V.: x = 2", color = "tomato",   size = 3.2) +
  annotate("text", x = 5.0, y = 2.6, label = "A.H.: y = 2", color = "steelblue", size = 3.2) +
  labs(title = "f(x) = 1/(x-2) + 2", x = "x", y = "f(x)") + base

p1 + p2

1.5 Continuidade

Uma função \(f\) é contínua em \(x = a\) quando três condições são satisfeitas simultaneamente:

  1. \(f(a)\) existe (o ponto está no domínio).
  2. \(\displaystyle\lim_{x \to a} f(x)\) existe (os limites laterais coincidem).
  3. \(\displaystyle\lim_{x \to a} f(x) = f(a)\) (o limite coincide com o valor da função).

Se qualquer uma dessas condições falhar, \(f\) é descontínua em \(x = a\).

1.5.1 Tipos de Descontinuidade

Descontinuidade removível: \(\lim_{x \to a} f(x)\) existe, mas \(\neq f(a)\) (ou \(f(a)\) não existe). O “buraco” pode ser preenchido.

Descontinuidade de salto: os limites laterais existem mas são diferentes. O gráfico “salta” em \(x = a\).

Descontinuidade essencial: pelo menos um dos limites laterais é infinito (assíntota vertical).

library(ggplot2)
library(patchwork)

base <- theme_minimal(base_size = 11)

# 1. Removível: f(x) = (x^2-4)/(x-2) com f(2)=1
x_rem <- seq(-1, 4, length.out = 400)
y_rem <- ifelse(abs(x_rem - 2) > 0.01, (x_rem^2 - 4)/(x_rem - 2), NA)
p1 <- ggplot(data.frame(x = x_rem, y = y_rem), aes(x, y)) +
  geom_line(color = "#440154", linewidth = 1, na.rm = TRUE) +
  geom_point(aes(x = 2, y = 4), color = "#440154", size = 3,
             shape = 21, fill = "white", stroke = 1.5) +
  geom_point(aes(x = 2, y = 1), color = "#FDE725", size = 3) +
  geom_hline(yintercept = 0, color = "gray80") +
  geom_vline(xintercept = 0, color = "gray80") +
  labs(title = "Removível", x = "x", y = "f(x)") + base

# 2. Salto
x2a <- seq(-2, 1,   length.out = 200)
x2b <- seq(1,  3.5, length.out = 200)
df2 <- rbind(
  data.frame(x = x2a, y = 3 + x2a),
  data.frame(x = x2b, y = 3 - x2b)
)
p2 <- ggplot(df2, aes(x, y)) +
  geom_line(aes(group = y > 2), color = "#31688e", linewidth = 1) +
  geom_point(aes(x = 1, y = 4), color = "#31688e", size = 3) +
  geom_point(aes(x = 1, y = 2), color = "#31688e", size = 3,
             shape = 21, fill = "white", stroke = 1.5) +
  geom_hline(yintercept = 0, color = "gray80") +
  geom_vline(xintercept = 0, color = "gray80") +
  labs(title = "Salto", x = "x", y = "f(x)") + base

# 3. Essencial (assíntota vertical)
xn3 <- seq(-3, -0.1, length.out = 200)
xp3 <- seq(0.1, 3,   length.out = 200)
df3 <- rbind(
  data.frame(x = xn3, y = 1/xn3),
  data.frame(x = xp3, y = 1/xp3)
)
p3 <- ggplot(df3, aes(x, y)) +
  geom_line(color = "#35b779", linewidth = 1) +
  geom_vline(xintercept = 0, linetype = "dashed", color = "tomato") +
  coord_cartesian(ylim = c(-8, 8)) +
  geom_hline(yintercept = 0, color = "gray80") +
  labs(title = "Essencial (A.V.)", x = "x", y = "f(x)") + base

p1 + p2 + p3

1.5.2 Propriedades da Continuidade

Se \(f\) e \(g\) são contínuas em \(x = a\), então também são contínuas em \(a\):

  • Soma e diferença: \(f(x) \pm g(x)\).
  • Produto: \(f(x) \cdot g(x)\).
  • Quociente: \(f(x)/g(x)\), desde que \(g(a) \neq 0\).
  • Composição: \(f(g(x))\), se \(g\) é contínua em \(a\) e \(f\) é contínua em \(g(a)\).

Classes de funções contínuas em todo o seu domínio: polinômios, funções racionais (exceto zeros do denominador), exponenciais, logarítmicas, trigonométricas.

1.5.3 Continuidade e Diferenciabilidade

Uma conexão fundamental com derivadas: diferenciabilidade implica continuidade, mas a recíproca é falsa.

Teorema. Se \(f'(a)\) existe, então \(f\) é contínua em \(a\).

Contraexemplo. \(f(x) = |x|\) é contínua em \(x = 0\), mas não é diferenciável ali — os limites laterais da razão incremental divergem:

\[\lim_{h \to 0^+} \frac{|h|}{h} = 1 \neq -1 = \lim_{h \to 0^-} \frac{|h|}{h}.\]

library(ggplot2)

x <- seq(-3, 3, length.out = 400)
ggplot(data.frame(x = x, y = abs(x)), aes(x, y)) +
  geom_line(color = "#440154", linewidth = 1.1) +
  geom_point(aes(x = 0, y = 0), color = "#FDE725", size = 3) +
  geom_hline(yintercept = 0, color = "gray80") +
  geom_vline(xintercept = 0, color = "gray80") +
  annotate("text", x = 1.5, y = 0.4, label = "ponto angular\n(não diferenciável)",
           color = "#FDE725", size = 3.5, fontface = "bold") +
  labs(x = "x", y = "f(x)", title = expression(f(x) == "|x|")) +
  theme_minimal(base_size = 12)
