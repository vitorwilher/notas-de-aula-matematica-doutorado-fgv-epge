1 Funções Exponenciais

1.1 Motivação Econômica

A função exponencial \(e^{rt}\) é o bloco de construção de praticamente toda dinâmica econômica contínua. Ela aparece sempre que o crescimento (ou decaimento) é proporcional ao tamanho atual da variável.

Crescimento do PIB. Se a taxa de crescimento do produto é constante \(g\), então \(Y(t) = Y_0 e^{gt}\). Após \(T\) anos, o PIB multiplica-se por \(e^{gT}\). Para \(g = 2\%\) ao ano e \(T = 35\) anos, \(e^{0.02 \times 35} \approx 2\) — a regra do número 70 (o produto dobra em \(70/g\) anos) é uma aproximação dessa fórmula.

Desconto contínuo e valor presente. O fator de desconto para um fluxo recebido no instante \(t\), a uma taxa de juros \(r\), é \(e^{-rt}\). O valor presente de um fluxo perpétuo \(\pi\) é \(\int_0^\infty \pi e^{-rt}\,dt = \pi/r\) — a fórmula da perpetuidade, obtida por integração da exponencial.

Capitalização contínua. Um capital \(K_0\) aplicado à taxa \(r\) composta continuamente cresce como \(K(t) = K_0 e^{rt}\). No limite da capitalização discreta com \(n \to \infty\) períodos por ano: \(K_0(1 + r/n)^{nt} \to K_0 e^{rt}\).

Depreciação e amortização. O estoque de capital deprecia-se à taxa \(\delta\): \(K(t) = K_0 e^{-\delta t}\). O mesmo modelo descreve a depreciação de qualquer ativo, o decaimento de dívidas, ou a queda de preços de bens duráveis.

Modelo de inflação de Cagan. A demanda por moeda no modelo de Cagan é \(m - p = -\alpha \dot{p}\), cuja solução estável envolve \(e^{-t/\alpha}\). A estabilidade do equilíbrio inflacionário depende do sinal do expoente.

1.2 Função Exponencial Natural \(e^x\)

1.2.1 Definição como Inversa do Logaritmo

Definimos \(\ln x = \displaystyle\int_1^x \dfrac{dt}{t}\) para \(x > 0\). Como \((\ln x)' = \dfrac{1}{x} > 0\), a função logaritmo é estritamente crescente e possui inversa. Definimos:

\[\exp(x) = e^x := (\ln)^{-1}(x), \quad x \in \mathbb{R}.\]

Ou seja, \(e^x = y \iff x = \ln y\), e as composições inversas dão:

\[\ln(e^x) = x \quad \forall x \in \mathbb{R}, \qquad e^{\ln x} = x \quad \forall x > 0.\]

Domínio e imagem:

\[\mathrm{Dom}(e^x) = \mathbb{R}, \qquad \mathrm{Im}(e^x) = \mathbb{R}_+ = (0, +\infty).\]

O número \(e\). A constante \(e \approx 2{,}71828\ldots\) é o único número real tal que \(\ln e = 1\), ou equivalentemente:

\[e = \exp(1), \qquad \ln e = \int_1^e \frac{dt}{t} = 1.\]

1.2.2 Propriedades de \(e^x\)

Para quaisquer \(a, b \in \mathbb{R}\):

Propriedade Fórmula
Produto \(e^a \cdot e^b = e^{a+b}\)
Quociente \(\dfrac{e^a}{e^b} = e^{a-b}\)
Potência \((e^a)^b = e^{ab}\)
Neutro \(e^0 = 1\)

Prova de \(e^a \cdot e^b = e^{a+b}\): Sejam \(A = e^a\) e \(B = e^b\), de modo que \(\ln A = a\) e \(\ln B = b\). Usando \(\ln(AB) = \ln A + \ln B = a + b\) e aplicando \(\exp\) em ambos os lados: \(AB = e^{a+b}\). \(\blacksquare\)

1.2.3 Derivada e Antiderivada

Derivada. Pelo Teorema da Função Inversa aplicado a \(f(x) = \ln x\):

\[\frac{d}{dx}e^x = \frac{1}{(\ln y)'}\bigg|_{y = e^x} = \frac{1}{1/e^x} = e^x.\]

A exponencial natural é a única função (a menos de múltiplos) igual à sua própria derivada.

Regra da cadeia. Para \(h(x) = e^{g(x)}\):

\[h'(x) = e^{g(x)} \cdot g'(x).\]

Antiderivada:

\[\int e^x\,dx = e^x + C.\]

Mais geralmente, se \(u = g(x)\):

\[\int e^{g(x)} g'(x)\,dx = e^{g(x)} + C.\]

1.2.4 Exemplos — Derivadas

Exemplo 1: \(f(x) = e^{x^2}\)

\[f'(x) = e^{x^2} \cdot 2x = 2x\,e^{x^2}.\]

Exemplo 2: \(f(x) = e^{1/x^2} = e^{x^{-2}}\)

\[f'(x) = e^{x^{-2}} \cdot (-2x^{-3}) = -\frac{2}{x^3}\,e^{1/x^2}.\]

Exemplo 3: \(y = \ln\!\bigl(e^x + e^{-x}\bigr)\)

\[y' = \frac{e^x - e^{-x}}{e^x + e^{-x}}.\]

Exemplo 4: \(y = \ln\!\left(\dfrac{e^{4x}-1}{e^{4x}+1}\right) = \ln(e^{4x}-1) - \ln(e^{4x}+1)\)

\[y' = \frac{4e^{4x}}{e^{4x}-1} - \frac{4e^{4x}}{e^{4x}+1} = \frac{4e^{4x}\bigl[(e^{4x}+1)-(e^{4x}-1)\bigr]}{(e^{4x}-1)(e^{4x}+1)} = \frac{8e^{4x}}{e^{8x}-1}.\]

1.2.5 Exemplo — Antiderivada

Exemplo 5: \(\displaystyle\int \frac{e^{\sqrt{x}}}{\sqrt{x}}\,dx\)

Fazemos \(u = \sqrt{x}\), \(du = \dfrac{dx}{2\sqrt{x}}\), logo \(\dfrac{dx}{\sqrt{x}} = 2\,du\):

\[\int \frac{e^{\sqrt{x}}}{\sqrt{x}}\,dx = 2\int e^u\,du = 2e^u + C = 2e^{\sqrt{x}} + C.\]

Exemplo 6: \(\displaystyle\int x^2 e^{2x^3}\,dx\)

Fazemos \(u = 2x^3\), \(du = 6x^2\,dx\), logo \(x^2\,dx = du/6\):

\[\int x^2 e^{2x^3}\,dx = \frac{1}{6}\int e^u\,du = \frac{1}{6}e^{2x^3} + C.\]

1.3 Segunda Definição de \(e\)

Além de \(e = \exp(1)\), existe uma definição via limite:

\[e = \lim_{h \to 0}(1+h)^{1/h} \approx 2{,}71828\ldots\]

Derivação. Como \(f(x) = \ln x\) e \(f'(1) = 1\):

\[1 = f'(1) = \lim_{h \to 0}\frac{\ln(1+h) - \ln 1}{h} = \lim_{h \to 0}\frac{1}{h}\ln(1+h) = \lim_{h \to 0}\ln(1+h)^{1/h}.\]

Como \(\exp\) é contínua: \(\exp(1) = \exp\!\left(\lim_{h\to0}\ln(1+h)^{1/h}\right) = \lim_{h\to0}(1+h)^{1/h}\).

Logo \(e = \lim_{h\to0}(1+h)^{1/h}\). \(\blacksquare\)

1.4 Diferenciação Logarítmica — \(f(x) = x^x\)

Para funções da forma \(f(x) = x^x\) (base e expoente variáveis), a regra da potência \(nx^{n-1}\) não se aplica diretamente. Usamos a técnica de diferenciação logarítmica:

\[f(x) = x^x = e^{\ln x^x} = e^{x\ln x}.\]

Derivando pela regra da cadeia:

\[f'(x) = e^{x\ln x}\cdot\frac{d}{dx}(x\ln x) = x^x\left(\ln x + x\cdot\frac{1}{x}\right) = x^x(\ln x + 1).\]

Alternativa via ln implícito. Tome \(\ln f(x) = x\ln x\). Diferenciando:

\[\frac{f'(x)}{f(x)} = \ln x + 1 \;\Rightarrow\; f'(x) = f(x)(\ln x+1) = x^x(\ln x+1).\]

1.5 Funções Exponenciais em Outras Bases

1.5.1 Definição e Derivada

Para \(a > 0\), \(a \neq 1\), definimos:

\[a^x := e^{x\ln a}.\]

Derivada. Pela regra da cadeia:

\[\frac{d}{dx}a^x = e^{x\ln a}\cdot\ln a = a^x\ln a.\]

Mais geralmente, para \(f(x) = a^{g(x)}\):

\[f'(x) = a^{g(x)}\,g'(x)\,\ln a.\]

Antiderivada:

\[\int a^x\,dx = \frac{a^x}{\ln a} + C.\]

Verificação: \(\left(\dfrac{a^x}{\ln a}\right)' = \dfrac{a^x\ln a}{\ln a} = a^x\). \(\checkmark\)

1.5.2 Propriedades de \(a^x\)

Propriedade Fórmula
Produto \(a^x\cdot a^y = a^{x+y}\)
Quociente \(a^x/a^y = a^{x-y}\)
Potência \((a^x)^y = a^{xy}\)
Produto de bases \((ab)^x = a^x b^x\)
Neutro \(a^0 = 1\)

1.5.3 Exemplos

Exemplo 7: \(f(x) = 6^{-3x}\)

\[f'(x) = 6^{-3x}\cdot(-3)\cdot\ln 6 = -3\ln 6\cdot 6^{-3x}.\]

Exemplo 8: \(f(x) = 10^{x^2-2x}\)

\[f'(x) = 10^{x^2-2x}\cdot(2x-2)\cdot\ln 10 = (2x-2)\ln 10\cdot 10^{x^2-2x}.\]

Exemplo 9: \(f(x) = 3^{x^2}\)

\[f'(x) = 3^{x^2}\cdot 2x\cdot\ln 3.\]

Exemplo 10: \(\displaystyle\int 3^{2x}\,dx\)

Fazemos \(u = 2x\), \(du = 2\,dx\):

\[\int 3^{2x}\,dx = \frac{1}{2}\int 3^u\,du = \frac{1}{2}\cdot\frac{3^u}{\ln 3}+C = \frac{3^{2x}}{2\ln 3}+C.\]

1.6 Logaritmo em Outras Bases e Mudança de Base

1.6.1 Definição

Para \(a > 0\), \(a \neq 1\), o logaritmo na base \(a\) de \(x > 0\) é:

\[y = \log_a x \iff x = a^y.\]

A fórmula de mudança de base expressa \(\log_a x\) em termos do logaritmo natural:

\[\log_a x = \frac{\ln x}{\ln a}.\]

Derivação: De \(x = a^y\) tomamos \(\ln\) em ambos os lados: \(\ln x = y\ln a\), logo \(y = \dfrac{\ln x}{\ln a}\).

Exemplos: \(\log_e e = \dfrac{\ln e}{\ln e} = 1\); \(\quad \log_{10} 1000 = \dfrac{\ln 1000}{\ln 10} = 3\).

1.6.2 Derivada

\[\frac{d}{dx}\log_a x = \frac{d}{dx}\frac{\ln x}{\ln a} = \frac{1}{x\ln a}.\]

Para \(h(x) = \log_a g(x)\):

\[h'(x) = \frac{g'(x)}{g(x)\,\ln a}.\]
