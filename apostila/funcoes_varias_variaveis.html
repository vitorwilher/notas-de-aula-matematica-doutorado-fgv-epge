1 Funções de Várias Variáveis Reais

1.1 Motivação Econômica

Na matemática do ensino médio e no cálculo univariado, estudamos funções de uma única variável real: \(y = f(x)\), em que um número real \(x\) é mapeado em um número real \(y\). Em Economia, porém, quase todas as relações de interesse dependem de múltiplos fatores simultaneamente — o consumo depende da renda e dos preços; a utilidade depende das quantidades de vários bens; o produto depende do capital, do trabalho e da tecnologia. Para modelar essas relações, precisamos generalizar o conceito de função para o caso de várias variáveis.

1.1.1 Definição Formal

Uma função de várias variáveis reais é uma regra que associa a cada vetor \((x_1, x_2, \ldots, x_n) \in \mathbb{R}^n\) um único valor real \(z \in \mathbb{R}\). Formalmente:

\[ f : D \subseteq \mathbb{R}^n \longrightarrow \mathbb{R}, \quad (x_1, x_2, \ldots, x_n) \mapsto z = f(x_1, x_2, \ldots, x_n). \]

Os elementos do domínio \(D\) são vetores (ou pontos) em \(\mathbb{R}^n\), e o valor \(z\) é um escalar real. O caso mais comum em Economia é \(n = 2\), com duas variáveis independentes \(x\) e \(y\):

\[ f : D \subseteq \mathbb{R}^2 \longrightarrow \mathbb{R}, \quad (x, y) \mapsto z = f(x, y). \]

O gráfico de \(f\) neste caso é uma superfície no espaço tridimensional \(\mathbb{R}^3\).

1.1.2 Conceitos Fundamentais

  • Domínio \(D\): conjunto de todos os pontos \((x_1,\ldots,x_n)\) para os quais \(f\) está definida.
  • Imagem \(\mathrm{Im}(f)\): conjunto de todos os valores \(z = f(x_1,\ldots,x_n)\) assumidos por \(f\).
  • Gráfico: conjunto \(\{(x_1,\ldots,x_n, z) \in \mathbb{R}^{n+1} : z = f(x_1,\ldots,x_n),\; (x_1,\ldots,x_n) \in D\}\).

1.1.3 Exemplo Simples: Custo Total

Considere uma firma que produz um bem usando dois insumos com preços \(w_1\) e \(w_2\), adquirindo quantidades \(x_1\) e \(x_2\). O custo total é:

\[ C(x_1, x_2) = w_1 x_1 + w_2 x_2. \]

Fixando \(w_1 = 3\) e \(w_2 = 5\), temos \(C(x_1, x_2) = 3x_1 + 5x_2\). Alguns valores:

\(x_1\) \(x_2\) \(C(x_1, x_2)\)
\(0\) \(0\) \(0\)
\(2\) \(0\) \(6\)
\(0\) \(2\) \(10\)
\(2\) \(3\) \(21\)
\(4\) \(4\) \(32\)

O domínio natural é \(D = \mathbb{R}^2_+ = \{(x_1, x_2) : x_1 \geq 0,\; x_2 \geq 0\}\) e a imagem é \([0, +\infty)\). As curvas de nível \(C(x_1, x_2) = \bar{C}\) são retas no plano \((x_1, x_2)\) — as isocustas da teoria microeconômica:

\[ x_2 = \frac{\bar{C}}{w_2} - \frac{w_1}{w_2}\,x_1 = \frac{\bar{C}}{5} - \frac{3}{5}\,x_1. \]

1.2 A Função de Produção

1.2.1 Motivação

Um exemplo clássico em Economia é a função de produção, que descreve a quantidade máxima de produto \(Q\) que uma firma pode obter combinando dois insumos: capital \(K\) e trabalho \(L\):

\[ f : \mathbb{R}^2_{+} \longrightarrow \mathbb{R}_{+}, \quad (K, L) \mapsto Q = f(K, L). \]

1.2.2 A Função Cobb-Douglas

A especificação mais utilizada na literatura é a Cobb-Douglas:

\[ \boxed{Q = f(K, L) = A \, K^{\alpha} \, L^{\beta},} \]

onde:

Parâmetro Interpretação
\(A > 0\) Produtividade total dos fatores (PTF)
\(\alpha \in (0,1)\) Elasticidade do produto em relação ao capital
\(\beta \in (0,1)\) Elasticidade do produto em relação ao trabalho

1.2.3 Capital Fixo: \(f\) como Função de \(L\)

Fixando \(K = \bar{K}\), a Cobb-Douglas se reduz a uma função de uma única variável:

\[ Q(L) = \underbrace{A\,\bar{K}^{\alpha}}_{\text{constante}}\; L^{\beta}, \]

que é uma função de potência em \(L\). Para \(\beta \in (0,1)\), \(Q\) é crescente e côncava — refletindo o produto marginal decrescente do trabalho.

library(ggplot2)

A <- 1; alpha <- 0.4; beta <- 0.6
L <- seq(0.01, 10, length.out = 300)

# Três níveis de capital fixo
df <- do.call(rbind, lapply(c(1, 2, 4), function(K) {
  data.frame(L = L, Q = A * K^alpha * L^beta, K = paste0("K = ", K))
}))

ggplot(df, aes(x = L, y = Q, color = K)) +
  geom_line(linewidth = 1) +
  labs(
    x = expression(L ~ "(trabalho)"),
    y = expression(Q ~ "(produto)"),
    title = expression(Q == A ~ bar(K)^alpha ~ L^beta),
    color = "Capital fixo"
  ) +
  theme_minimal(base_size = 13) +
  scale_color_viridis_d(end = 0.85)

1.2.4 Retornos de Escala

Multiplicando todos os insumos por um fator \(\lambda > 0\):

\[ f(\lambda K,\, \lambda L) = A\,(\lambda K)^{\alpha}\,(\lambda L)^{\beta} = \lambda^{\alpha+\beta}\, A K^{\alpha} L^{\beta} = \lambda^{\alpha+\beta}\, f(K,L). \]

  • Se \(\alpha + \beta = 1\): retornos constantes de escala.
  • Se \(\alpha + \beta > 1\): retornos crescentes de escala.
  • Se \(\alpha + \beta < 1\): retornos decrescentes de escala.

Para visualizar, fixamos \((K_0, L_0) = (1, 1)\) e escalamos os insumos por \(\lambda \in [1, 4]\). O produto resultante é \(Q(\lambda) = \lambda^{\alpha+\beta} \cdot f(K_0, L_0)\). Quando \(\lambda\) dobra, o produto mais do que dobra (crescentes), dobra exatamente (constantes) ou menos que dobra (decrescentes).

library(ggplot2)

lambda <- seq(1, 4, length.out = 300)
A <- 1; K0 <- 1; L0 <- 1

casos <- list(
  list(label = "Crescentes (α+β = 1.3)", ab = 1.3),
  list(label = "Constantes (α+β = 1.0)", ab = 1.0),
  list(label = "Decrescentes (α+β = 0.7)", ab = 0.7)
)

df <- do.call(rbind, lapply(casos, function(c) {
  Q0 <- A * K0^(c$ab * 0.5) * L0^(c$ab * 0.5)
  data.frame(lambda = lambda, Q = lambda^c$ab * Q0, caso = c$label)
}))

ggplot(df, aes(x = lambda, y = Q, color = caso)) +
  geom_line(linewidth = 1) +
  geom_abline(slope = 1, intercept = 0, linetype = "dashed",
              color = "gray50", linewidth = 0.5) +
  annotate("text", x = 3.8, y = 3.6, label = "Q = λ (ref.)",
           color = "gray50", size = 3.2, hjust = 1) +
  labs(
    x = expression(lambda ~ "(fator de escala)"),
    y = expression(Q(lambda)),
    title = "Retornos de Escala na Cobb-Douglas",
    color = NULL
  ) +
  theme_minimal(base_size = 13) +
  scale_color_viridis_d(end = 0.85) +
  theme(legend.position = "bottom")

1.3 Curvas de Nível

1.3.1 Definição

Dada \(f : D \subseteq \mathbb{R}^2 \to \mathbb{R}\), a curva de nível de valor \(c \in \mathbb{R}\) é o conjunto:

\[ \mathcal{C}_c = \bigl\{(x, y) \in D : f(x, y) = c\bigr\}. \]

Geometricamente, \(\mathcal{C}_c\) é a projeção no plano \(xy\) da interseção do gráfico de \(f\) com o plano horizontal \(z = c\). O conjunto de todas as curvas de nível, para diferentes valores de \(c\), é chamado de mapa de contorno de \(f\).

1.3.2 Interpretação Econômica: Isoquantas

Na função de produção \(Q = f(K, L)\), as curvas de nível são as isoquantas — combinações de capital e trabalho que geram o mesmo nível de produto:

\[ \mathcal{C}_{\bar{Q}} = \bigl\{(K, L) : A\,K^{\alpha}\,L^{\beta} = \bar{Q}\bigr\}. \]

Isolando \(L\):

\[ L = \left(\frac{\bar{Q}}{A\,K^{\alpha}}\right)^{1/\beta} = \left(\frac{\bar{Q}}{A}\right)^{1/\beta} K^{-\alpha/\beta}. \]

Para \(\alpha, \beta > 0\), a isoquanta é decrescente e convexa no plano \((K, L)\) — a firma pode trocar capital por trabalho mantendo o produto constante, mas a substituição é cada vez mais custosa.

1.3.3 Circunferências e a Equação Geral

A equação geral de uma circunferência de raio \(R\) centrada no ponto \((a, b)\) é:

\[ \boxed{(x - a)^2 + (y - b)^2 = R^2.} \]

No caso de \(f(x, y) = x^2 + y^2\), a curva de nível \(f(x,y) = c\) é:

\[ x^2 + y^2 = c \quad \Longleftrightarrow \quad (x - 0)^2 + (y - 0)^2 = (\sqrt{c}\,)^2, \]

ou seja, uma circunferência com \(a = 0\), \(b = 0\) e \(R = \sqrt{c}\) — centrada na origem. À medida que \(c\) cresce, o raio \(R = \sqrt{c}\) aumenta, e as circunferências se afastam da origem.

1.3.4 Exemplos Gráficos

A forma das curvas de nível revela a geometria da função. Três casos ilustrativos:

Função Curvas de nível Geometria
\(f(x,y) = x^2 + y^2\) \(x^2 + y^2 = c\) Circunferências
\(f(x,y) = x^2 - y^2\) \(x^2 - y^2 = c\) Hipérboles
\(f(x,y) = K^{0.4} L^{0.6}\) \(K^{0.4} L^{0.6} = c\) Isoquantas côncavas

1.4 Exemplo: \(f(x, y) = x^2 + y^2\)

Considere a função \(f : \mathbb{R}^2 \longrightarrow \mathbb{R}\) definida por:

\[ f(x, y) = x^2 + y^2. \]

1.4.1 Domínio e Imagem

  • Domínio: \(D = \mathbb{R}^2\), pois a expressão está definida para todo \((x, y) \in \mathbb{R}^2\).
  • Imagem: \(\mathrm{Im}(f) = [0, +\infty)\), pois \(x^2 + y^2 \geq 0\) para todo \((x,y)\), com igualdade apenas na origem.

1.4.2 Curvas de Nível

As curvas de nível de \(f\) são os conjuntos \(\{(x,y) \in \mathbb{R}^2 : f(x,y) = c\}\) para \(c \geq 0\):

\[ x^2 + y^2 = c. \]

Para \(c > 0\), cada curva de nível é uma circunferência de raio \(\sqrt{c}\) centrada na origem; para \(c = 0\), é o ponto \((0,0)\).

1.4.3 Avaliação em Pontos

Alguns valores ilustrativos:

\((x, y)\) \(f(x, y)\)
\((0, 0)\) \(0\)
\((1, 0)\) \(1\)
\((1, 1)\) \(2\)
\((3, 4)\) \(25\)

1.4.4 Gráfico

O gráfico de \(f\) é um paraboloide elíptico (circular) com vértice na origem, que se abre para cima:

\[ z = x^2 + y^2, \quad z \geq 0. \]

1.4.4.1 Com R (ggplot2)

library(ggplot2)

grid <- expand.grid(
  x = seq(-3, 3, length.out = 200),
  y = seq(-3, 3, length.out = 200)
)
grid$z <- grid$x^2 + grid$y^2

ggplot(grid, aes(x = x, y = y, z = z)) +
  geom_contour_filled(breaks = c(0, 1, 2, 4, 6, 9), alpha = 0.85) +
  geom_contour(breaks = c(0, 1, 2, 4, 6, 9), color = "white", linewidth = 0.4) +
  scale_fill_viridis_d(name = expression(f(x,y))) +
  labs(
    x = expression(x),
    y = expression(y),
    title = expression(f(x, y) == x^2 + y^2)
  ) +
  coord_fixed() +
  theme_minimal(base_size = 13)

1.4.4.2 Com Python (matplotlib + numpy)

A lógica é idêntica à do R: construímos uma grade de pontos no plano \(xy\), avaliamos \(f\) em cada ponto e pedimos ao matplotlib que trace as curvas de nível.

import numpy as np
import matplotlib.pyplot as plt

# 1. Grade de pontos: 200 valores igualmente espaçados em [-3, 3] para cada eixo
x = np.linspace(-3, 3, 200)
y = np.linspace(-3, 3, 200)

# 2. meshgrid cria duas matrizes 200×200: X[i,j] = x[j] e Y[i,j] = y[i]
#    Assim conseguimos avaliar f em todos os pares (x, y) de uma vez
X, Y = np.meshgrid(x, y)

# 3. Avalia f(x,y) = x² + y² em cada ponto da grade (operação vetorizada)
Z = X**2 + Y**2

# 4. Plota as curvas de nível preenchidas nos mesmos níveis usados no R
niveis = [0, 1, 2, 4, 6, 9]

fig, ax = plt.subplots(figsize=(5, 5))
cf = ax.contourf(X, Y, Z, levels=niveis, cmap="viridis", alpha=0.85)
ax.contour(X, Y, Z, levels=niveis, colors="white", linewidths=0.5)
fig.colorbar(cf, ax=ax, label=r"$f(x,y)$")
<matplotlib.colorbar.Colorbar object at 0x13419e270>
ax.set_xlabel("$x$")
ax.set_ylabel("$y$")
ax.set_title(r"$f(x, y) = x^2 + y^2$")
ax.set_aspect("equal")
plt.tight_layout()
plt.show()

1.5 Exemplo: \(f(x, y) = \dfrac{1}{x - y^2}\)

1.5.1 Domínio

A função não está definida quando o denominador se anula. Portanto:

\[ D = \bigl\{(x, y) \in \mathbb{R}^2 : x - y^2 \neq 0\bigr\} = \mathbb{R}^2 \setminus \{x = y^2\}. \]

O conjunto excluído, \(x = y^2\), é uma parábola no plano \(xy\). À esquerda dela (\(x < y^2\)), \(f < 0\); à direita (\(x > y^2\)), \(f > 0\).

1.5.2 Imagem

  • Quando \(x - y^2 \to 0^+\), \(f \to +\infty\).
  • Quando \(x - y^2 \to 0^-\), \(f \to -\infty\).
  • \(f\) pode assumir qualquer valor real exceto zero (pois \(1/(x-y^2) = 0\) não tem solução).

\[ \mathrm{Im}(f) = \mathbb{R} \setminus \{0\}. \]

1.5.3 Curvas de Nível

Igualando \(f(x,y) = c \neq 0\):

\[ \frac{1}{x - y^2} = c \quad \Longrightarrow \quad x - y^2 = \frac{1}{c} \quad \Longrightarrow \quad \boxed{x = y^2 + \frac{1}{c}.} \]

Cada curva de nível é uma parábola com eixo horizontal, deslocada da parábola-fronteira \(x = y^2\) por \(1/c\):

  • \(c > 0\): parábola deslocada para a direita (\(1/c > 0\)).
  • \(c < 0\): parábola deslocada para a esquerda (\(1/c < 0\)).
  • À medida que \(|c| \to \infty\), as curvas se aproximam da fronteira \(x = y^2\).

1.5.4 Gráfico

library(ggplot2)

N   <- 400
grd <- expand.grid(x = seq(-1, 4, length.out = N),
                   y = seq(-3, 3, length.out = N))

# Mascara pontos muito próximos da fronteira (evita artefatos numéricos)
denom       <- grd$x - grd$y^2
grd$z       <- ifelse(abs(denom) > 0.05, 1 / denom, NA)

niveis_inv  <- c(-4, -2, -1, -0.5, 0.5, 1, 2, 4)

# Parábola-fronteira para referência
parabola <- data.frame(y = seq(-3, 3, length.out = 300))
parabola$x <- parabola$y^2

ggplot(grd, aes(x = x, y = y, z = z)) +
  geom_contour_filled(breaks = niveis_inv, alpha = 0.80, show.legend = FALSE,
                      na.rm = TRUE) +
  geom_contour(breaks = niveis_inv, color = "white", linewidth = 0.35,
               na.rm = TRUE) +
  geom_label(
    data = data.frame(
      x     = 1 / niveis_inv,           # x = 1/c quando y = 0
      y     = rep(0, length(niveis_inv)),
      label = as.character(niveis_inv)
    ),
    aes(x = x, y = y, label = label), inherit.aes = FALSE,
    size = 2.8, fill = "white", color = "black",
    linewidth = 0.2, label.padding = unit(0.15, "lines")
  ) +
  geom_line(data = parabola, aes(x = x, y = y), inherit.aes = FALSE,
            color = "red", linewidth = 0.7, linetype = "dashed") +
  annotate("text", x = 3.5, y = 1.6, label = "x == y^2", parse = TRUE,
           color = "red", size = 3.5) +
  scale_fill_viridis_d() +
  coord_cartesian(xlim = c(-1, 4), ylim = c(-3, 3)) +
  labs(
    x        = expression(x),
    y        = expression(y),
    title    = expression(f(x, y) == frac(1, x - y^2)),
    subtitle = expression(x - y^2 == frac(1, c))
  ) +
  theme_minimal(base_size = 13)

1.6 Exemplo: \(f(x, y) = \dfrac{1}{x^2 + y^2}\)

1.6.1 Domínio e Imagem

O denominador se anula apenas na origem, portanto:

\[ D = \mathbb{R}^2 \setminus \{(0,0)\}. \]

Como \(x^2 + y^2 > 0\) em todo o domínio, \(f > 0\) em todo ponto. Além disso:

  • Quando \((x,y) \to (0,0)\): \(x^2+y^2 \to 0^+\), logo \(f \to +\infty\).
  • Quando \(\|(x,y)\| \to \infty\): \(x^2+y^2 \to +\infty\), logo \(f \to 0^+\).

Portanto \(\mathrm{Im}(f) = (0, +\infty)\).

1.6.2 Curvas de Nível

Igualando \(f(x,y) = c > 0\):

\[ \frac{1}{x^2+y^2} = c \quad\Longrightarrow\quad x^2 + y^2 = \frac{1}{c}. \]

Comparando com a equação geral \((x-a)^2+(y-b)^2 = R^2\), temos \(a=0\), \(b=0\) e \(R^2 = 1/c\) — circunferências centradas na origem com raio decrescente em \(c\).

Contraste com \(f = x^2+y^2\): naquele exemplo \(R^2 = c\) cresce com \(c\); aqui \(R^2 = 1/c\) decresce — curvas de nível maiores correspondem a círculos menores, mais próximos da origem.

\(c\) \(R^2 = 1/c\)
\(0.25\) \(4\)
\(0.5\) \(2\)
\(1\) \(1\)
\(2\) \(0.5\)
\(4\) \(0.25\)

1.7 Derivadas Parciais

1.7.1 Definição

Dada \(f : D \subseteq \mathbb{R}^2 \to \mathbb{R}\), a derivada parcial de \(f\) em relação a \(x\) no ponto \((x_0, y_0)\) é definida pelo limite:

\[ \frac{\partial f}{\partial x}(x_0, y_0) = \lim_{h \to 0} \frac{f(x_0 + h,\, y_0) - f(x_0, y_0)}{h}, \]

e analogamente para \(y\):

\[ \frac{\partial f}{\partial y}(x_0, y_0) = \lim_{h \to 0} \frac{f(x_0,\, y_0 + h) - f(x_0, y_0)}{h}. \]

A ideia é simples: congela-se uma variável e diferencia-se em relação à outra, exatamente como na derivada univariada.

1.7.2 Notação

Para \(z = f(x, y)\), todas as notações abaixo são equivalentes:

\[ \frac{\partial f}{\partial x} = f_x = \partial_x f = D_x f = z_x. \]

1.7.3 Interpretação Geométrica

\(\dfrac{\partial f}{\partial x}(x_0, y_0)\) é a inclinação da curva obtida pela interseção do gráfico de \(f\) com o plano \(y = y_0\) — ou seja, a taxa de variação de \(f\) na direção do eixo \(x\), mantendo \(y\) fixo.

1.7.4 Cálculo Prático

Para calcular \(\partial f / \partial x\), trata-se \(y\) como constante e aplica-se as regras usuais de derivação. Analogamente para \(\partial f / \partial y\).

1.7.5 Exemplos

1.7.5.1 \(f(x, y) = x^2 + y^2\)

\[ \frac{\partial f}{\partial x} = 2x, \qquad \frac{\partial f}{\partial y} = 2y. \]

1.7.5.2 \(f(x, y) = x^2 - y^2\)

\[ \frac{\partial f}{\partial x} = 2x, \qquad \frac{\partial f}{\partial y} = -2y. \]

1.7.5.3 \(f(x, y) = \dfrac{1}{x - y^2}\)

Usando a regra da cadeia com \(u = x - y^2\):

\[ \frac{\partial f}{\partial x} = -\frac{1}{(x - y^2)^2}, \qquad \frac{\partial f}{\partial y} = \frac{2y}{(x - y^2)^2}. \]

1.7.5.4 Cobb-Douglas: \(f(K, L) = A K^{\alpha} L^{\beta}\)

\[ \frac{\partial f}{\partial K} = \alpha A K^{\alpha - 1} L^{\beta} = \frac{\alpha \, Q}{K}, \qquad \frac{\partial f}{\partial L} = \beta A K^{\alpha} L^{\beta - 1} = \frac{\beta \, Q}{L}. \]

Estas são o produto marginal do capital (\(PMgK\)) e o produto marginal do trabalho (\(PMgL\)) — a variação no produto decorrente de uma unidade adicional de cada insumo, mantendo o outro fixo.

1.7.6 Interpretação Econômica

Na Cobb-Douglas, \(\alpha\) e \(\beta\) têm uma interpretação direta via derivadas parciais:

\[ \frac{\partial \ln Q}{\partial \ln K} = \alpha, \qquad \frac{\partial \ln Q}{\partial \ln L} = \beta, \]

ou seja, \(\alpha\) e \(\beta\) são as elasticidades do produto em relação ao capital e ao trabalho, respectivamente. Um aumento de 1% em \(K\), mantendo \(L\) fixo, eleva \(Q\) em \(\alpha\)%.

1.7.7 Ilustração Gráfica

1.8 Derivadas Parciais de Segunda Ordem

1.8.1 Definição e Notação

Dada \(f : D \subseteq \mathbb{R}^2 \to \mathbb{R}\) com derivadas parciais de primeira ordem, as derivadas parciais de segunda ordem são obtidas diferenciando novamente. Para \(z = f(x,y)\), há quatro derivadas de segunda ordem, cada uma com notações equivalentes:

Derivadas puras:

\[ \frac{\partial^2 f}{\partial x^2} = f_{xx} = \frac{\partial}{\partial x}\!\left(\frac{\partial f}{\partial x}\right), \qquad \frac{\partial^2 f}{\partial y^2} = f_{yy} = \frac{\partial}{\partial y}\!\left(\frac{\partial f}{\partial y}\right). \]

Derivadas mistas (diferencia-se em relação a variáveis distintas):

\[ f_{yx} = \frac{\partial^2 f}{\partial y \, \partial x} = \frac{\partial}{\partial y}\!\left(\frac{\partial f}{\partial x}\right), \qquad f_{xy} = \frac{\partial^2 f}{\partial x \, \partial y} = \frac{\partial}{\partial x}\!\left(\frac{\partial f}{\partial y}\right). \]

Ordem de leitura: tanto nos subscritos quanto na notação de Leibniz, lê-se da esquerda para a direita. Assim, \(f_{yx} = \partial^2 f/\partial y\,\partial x\) significa diferenciar primeiro em \(y\) e depois em \(x\). O operador mais à esquerda é o aplicado por último — envolve o resultado da diferenciação anterior.

1.8.2 Teorema de Young

Enunciado: Seja \(f : D \subseteq \mathbb{R}^2 \to \mathbb{R}\). Se as derivadas mistas \(f_{xy}\) e \(f_{yx}\) existem e são contínuas em \((x_0, y_0)\), então:

\[ \boxed{f_{xy}(x_0, y_0) = f_{yx}(x_0, y_0).} \]

O que o teorema diz: a ordem de diferenciação é irrelevante — diferencia-se primeiro em \(x\) e depois em \(y\), ou vice-versa, e o resultado é o mesmo. Isso reduz o número de derivadas distintas de 4 para 3: \(f_{xx}\), \(f_{yy}\) e \(f_{xy} = f_{yx}\).

Por que a continuidade importa: sem ela, o teorema pode falhar. O contraexemplo clássico é:

\[ f(x,y) = \begin{cases} \dfrac{xy(x^2 - y^2)}{x^2 + y^2} & (x,y) \neq (0,0) \\ 0 & (x,y) = (0,0), \end{cases} \]

para a qual \(f_{xy}(0,0) = 1 \neq -1 = f_{yx}(0,0)\) — as mistas existem mas não são contínuas na origem.

Consequência prática: para as funções usuais em Economia (polinômios, exponenciais, logaritmos, potências), as derivadas mistas são contínuas em todo o domínio, de modo que \(f_{xy} = f_{yx}\) é garantido e a Hessiana é sempre simétrica.

Ou seja, a ordem de diferenciação não importa — resultado de grande utilidade prática.

1.8.3 A Matriz Hessiana

Todas as derivadas de segunda ordem de \(f(x,y)\) organizam-se na matriz Hessiana:

\[ H_f(x,y) = \begin{pmatrix} \dfrac{\partial^2 f}{\partial x^2} & \dfrac{\partial^2 f}{\partial x \, \partial y} \\[10pt] \dfrac{\partial^2 f}{\partial y \, \partial x} & \dfrac{\partial^2 f}{\partial y^2} \end{pmatrix}. \]

Pelo Teorema de Schwarz, \(H_f\) é simétrica sempre que as derivadas mistas são contínuas. A Hessiana captura a curvatura de \(f\) e é central nas condições de segunda ordem para otimização.

1.8.4 Exemplos

1.8.4.1 \(f(x, y) = x^2 + y^2\)

\[ f_x = 2x, \quad f_y = 2y, \]

\[ f_{xx} = 2, \quad f_{yy} = 2, \quad f_{xy} = f_{yx} = 0, \]

\[ H_f = \begin{pmatrix} 2 & 0 \\ 0 & 2 \end{pmatrix}. \]

Hessiana diagonal com entradas positivas: curvatura positiva em todas as direções — paraboloide convexo.

1.8.4.2 \(f(x, y) = x^2 - y^2\)

\[ f_{xx} = 2, \quad f_{yy} = -2, \quad f_{xy} = 0, \]

\[ H_f = \begin{pmatrix} 2 & 0 \\ 0 & -2 \end{pmatrix}. \]

Curvatura positiva em \(x\) e negativa em \(y\) — superfície de sela.

1.8.4.3 Cobb-Douglas: \(f(K, L) = A K^{\alpha} L^{\beta}\)

\[ f_{KK} = \alpha(\alpha-1)\,A K^{\alpha-2} L^{\beta}, \qquad f_{LL} = \beta(\beta-1)\,A K^{\alpha} L^{\beta-2}, \]

\[ f_{KL} = f_{LK} = \alpha\beta\,A K^{\alpha-1} L^{\beta-1}. \]

\[ H_f = \begin{pmatrix} \alpha(\alpha-1)\,AK^{\alpha-2}L^{\beta} & \alpha\beta\,AK^{\alpha-1}L^{\beta-1} \\[6pt] \alpha\beta\,AK^{\alpha-1}L^{\beta-1} & \beta(\beta-1)\,AK^{\alpha}L^{\beta-2} \end{pmatrix}. \]

Para \(\alpha, \beta \in (0,1)\): \(f_{KK} < 0\) e \(f_{LL} < 0\) — produto marginal decrescente de cada insumo.

1.8.4.4 \(f(x, y) = \ln(1 + x^2 + y^2)\)

Seja \(u = 1 + x^2 + y^2\). Pela regra da cadeia:

\[ f_x = \frac{2x}{1 + x^2 + y^2}, \qquad f_y = \frac{2y}{1 + x^2 + y^2}. \]

Diferenciando novamente pela regra do quociente:

\[ f_{xx} = \frac{2(1 + x^2 + y^2) - 2x \cdot 2x}{(1+x^2+y^2)^2} = \frac{2(1 - x^2 + y^2)}{(1+x^2+y^2)^2}, \]

\[ f_{yx} = \frac{\partial}{\partial y}\!\left(\frac{2x}{1+x^2+y^2}\right) = \frac{-4xy}{(1+x^2+y^2)^2}, \]

\[ f_{xy} = \frac{\partial}{\partial x}\!\left(\frac{2y}{1+x^2+y^2}\right) = \frac{-4xy}{(1+x^2+y^2)^2} = f_{yx}. \quad \checkmark \]

1.8.4.5 \(f(x, y) = \dfrac{1}{x^2 + y^2}\)

Seja \(r^2 = x^2 + y^2\). Escrevendo \(f = (x^2+y^2)^{-1}\):

\[ f_x = \frac{-2x}{(x^2+y^2)^2}, \qquad f_y = \frac{-2y}{(x^2+y^2)^2}. \]

\[ f_{xx} = \frac{-2(x^2+y^2)^2 + 2x \cdot 2(x^2+y^2)\cdot 2x}{(x^2+y^2)^4} = \frac{6x^2 - 2y^2}{(x^2+y^2)^3}, \]

\[ f_{yx} = \frac{\partial}{\partial y}\!\left(\frac{-2x}{(x^2+y^2)^2}\right) = \frac{8xy}{(x^2+y^2)^3}, \]

\[ f_{xy} = \frac{\partial}{\partial x}\!\left(\frac{-2y}{(x^2+y^2)^2}\right) = \frac{8xy}{(x^2+y^2)^3} = f_{yx}. \quad \checkmark \]

1.8.4.6 \(F(x, y) = x\,e^{x^2 - y^2}\)

Seja \(u = x^2 - y^2\). Pela regra do produto e da cadeia:

\[ F_x = e^{x^2-y^2} + x \cdot 2x\,e^{x^2-y^2} = e^{x^2-y^2}(1 + 2x^2), \]

\[ F_y = x \cdot (-2y)\,e^{x^2-y^2} = -2xy\,e^{x^2-y^2}. \]

\[ F_{xx} = 2x\,e^{x^2-y^2}(1+2x^2) + e^{x^2-y^2}\cdot 4x = 2x\,e^{x^2-y^2}(3 + 2x^2), \]

\[ F_{yx} = \frac{\partial}{\partial y}\!\left[e^{x^2-y^2}(1+2x^2)\right] = -2y(1+2x^2)\,e^{x^2-y^2}, \]

\[ F_{xy} = \frac{\partial}{\partial x}\!\left[-2xy\,e^{x^2-y^2}\right] = -2y\,e^{x^2-y^2} - 4x^2 y\,e^{x^2-y^2} = -2y(1+2x^2)\,e^{x^2-y^2} = F_{yx}. \quad \checkmark \]

1.8.5 Interpretação Econômica

Na Cobb-Douglas, \(f_{KK} < 0\) e \(f_{LL} < 0\) traduzem a lei dos rendimentos marginais decrescentes: cada unidade adicional de capital (ou trabalho), mantendo o outro fator fixo, contribui cada vez menos para o produto. A derivada mista \(f_{KL} > 0\) indica complementaridade: mais capital eleva o produto marginal do trabalho, e vice-versa.
